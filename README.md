# 30DayWebDev
# 30天网页开发学习计划，分为5个难度等级，每天一个练手项目：

## 【入门级】Day 1-5：HTML基础
个人简介页面、文章排版、图片画廊、表单设计、响应式布局
## 【初级】Day 6-10：CSS进阶
导航栏、卡片设计、动画效果、登录页面、响应式简历
## 【中级】Day 11-20：JavaScript基础
计算器、待办事项、颜色猜谜、图片轮播、模态框、天气查询、倒计时器、笔记应用、表单验证、滑动拼图
## 【高级】Day 21-25：现代前端技术
Vue.js待办列表、React天气应用、响应式网格、CSS动画、JSON数据展示
## 【专家级】Day 26-30：综合项目
个人博客、电商产品页、仪表盘、聊天界面、作品集网站
每个项目都包含详细说明、代码示例和AI辅助学习建议。
